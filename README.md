# KALMS — Website

Statische Website (HTML/CSS/JS), gehostet auf Vercel.
**Live:** https://kalms-orpin.vercel.app

## Bearbeiten (ohne Programme)
1. Datei hier auf GitHub öffnen (z. B. `index.html`), Stift-Symbol klicken.
2. Text ändern → **Commit changes**.
3. Vercel veröffentlicht die Änderung automatisch in ~20 Sekunden.

## Bilder
Fotos liegen in `assets/`. Neues Bild nach Namensschema ablegen, dann erscheint es automatisch
(siehe Bildnamen wie `hero-case-fog.jpg`, `truhe-1.jpg`, …).
